 # Heart Attack Prediction – Machine Learning Project
 Project Overview

This project analyzes a heart attack dataset and applies machine learning techniques to predict the presence of heart disease. The notebook includes data preprocessing, exploratory data analysis (EDA), visualization, and model training for classification tasks.

# Files Included

heart_attack.ipynb – Main Jupyter Notebook containing analysis, visualizations, and ML models

heart_attack.csv – Dataset used for training and evaluation

# Dataset Description

The dataset contains 918 patient records with 12 medical features such as age, blood pressure, cholesterol, ECG results, and exercise-related indicators.
The target variable is HeartDisease (1 = Disease, 0 = No Disease).
